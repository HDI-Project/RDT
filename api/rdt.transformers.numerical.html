Transformer for numerical data.

This transformer replaces integer values with their float equivalent. Non null float values are not modified.

Null values are replaced using a NullTransformer.

  • null_column (bool) â€“ Whether to create a new column to indicate which values were null or not. If None, only create a new column when the data contains null values.
